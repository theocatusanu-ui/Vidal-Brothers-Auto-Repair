## Setup

```bash
npm install
```

Optional: create a `.env` file in the project root if you use env-based config (e.g. `GEMINI_API_KEY`).

## Scripts

| Command        | Description              |
|----------------|--------------------------|
| `npm run dev`  | Start dev server         |
| `npm run build`| Production build         |
| `npm run preview` | Preview production build |
